## 📌 About This Repository

**dev-playground** is my personal sandbox where I experiment, prototype, and learn. This is not a production-ready codebase — it's a collection of experiments, half-finished ideas, and learning projects that document my journey as a developer.

Every file here represents something I tried, something I broke, and something I learned from.

---

## 🎯 What's Inside

### 1. **Web Development Experiments**
Small frontend experiments exploring HTML, CSS, and JavaScript concepts.

| Project | Description |
|---------|-------------|
| `landing-page-experiments/` | Various landing page layouts and designs |
| `css-animations/` | CSS keyframe animations and transitions |
| `javascript-demos/` | Vanilla JS interactive demos and utilities |
| `responsive-layouts/` | Mobile-first responsive design experiments |

### 2. **Backend & API Concepts**
Learning server-side development with Node.js and REST APIs.

| Project | Description |
|---------|-------------|
| `api-playground/` | REST API experiments with Express.js |
| `auth-demo/` | Basic authentication and session management |
| `database-connections/` | MongoDB and PostgreSQL connection examples |
| `webhooks/` | Webhook handlers and event-driven logic |

### 3. **AI Tools & LLM Integration**
Experiments with large language models and AI-powered workflows.

| Project | Description |
|---------|-------------|
| `llm-api-calls/` | Integration with Google AI Studio and other LLM APIs |
| `prompt-engineering/` | Prompt templates and optimization experiments |
| `ollama-local/` | Self-hosted LLM experiments with Ollama |
| `ai-chatbots/` | Simple chatbot implementations |

### 4. **Self-Hosted Workflows**
Setting up and managing local development environments.

| Project | Description |
|---------|-------------|
| `docker-experiments/` | Dockerfiles and container configurations |
| `local-server-setup/` | Self-hosted server configurations |
| `linux-scripts/` | Bash scripts for automation and DevOps tasks |
| `homelab-configs/` | Home server and self-hosting configurations |

### 5. **Minecraft & Gaming**
Technical experiments related to Minecraft modding and servers.

| Project | Description |
|---------|-------------|
| `mc-server-configs/` | Minecraft server configuration files |
| `mod-installations/` | Mod pack setups and custom configurations |
| `schematic-experiments/` | WorldEdit and schematic tools |
